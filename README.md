nulllab推出基于xc620芯片的开发板
芯片特点
- 蓝牙4.2
- ARM-CortexM0 32M主频,128K ram,256K flash
- 2路Uart(Uart0带流控)
- 8路10bit ADC
- 1路SPI
- 6路PWM
